# Todo Application - Phase I

A console-based todo application with in-memory storage, implemented in Python.

## Features

- Add new tasks with titles and optional descriptions
- View all tasks with their completion status
- Update existing tasks (title and description)
- Delete tasks by ID
- Mark tasks as complete/incomplete
- Menu-driven interface for easy navigation

## Requirements

- Python 3.13+

## Usage

Run the application:

```bash
python src/main.py
```

Follow the on-screen menu prompts to manage your tasks.

## Architecture

- Single-file Python application
- In-memory data storage (tasks are not persisted beyond runtime)
- Console-based user interface
- Menu-driven navigation with numeric input

## Phase I Scope

This is Phase I of the Evolution of Todo project, implementing basic task management functionality with:
- In-memory Python console application
- Single user
- No persistence beyond runtime
- Core CRUD operations for tasks
